# PwjLeetcodeSolution
Own leetcode solution summary

这是我自己的LeetCode的刷题记录，主要使用的Java，会根据不同的题型进行一个分类，每一个题会单独放在一个单独的包中。

同时这里也会更新一些算法和数据结构有关的内容。

# 主要算法
- 分治与递归
- 记忆优化
- 动态规划（空间的优化）
- 回溯算法
   - 字符串的所有组合与排列
- 双指针
   - 通过移动边界减少无用运算
- 排序算法
   - 堆排序
   - 归并排序
- 搜索算法
   - 二分查找
   - DFS和BFS
- 数据结构
   - Stack
   - Heap
   - 链表
   - 树
   - Hash表
- DFA
